**Breast Cancer Prediction System**
Name: Haset Wendesen

Department: Software Engineering

It predicts whether a tumor is benign or malignant based on selected medical features.
Two machine learning models are implemented:
   Logistic Regression
   Decision Tree Classifier
Models were trained using the Breast Cancer Wisconsin Dataset(kaggle)   
Models were saved using joblib
